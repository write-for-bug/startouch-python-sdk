# Installation

## Stable release

To install StarTouch Python SDK, run this command in your terminal:

```sh
uv add startouch-python-sdk
```

Or if you prefer to use `pip`:

```sh
pip install startouch-python-sdk
```

## From source

The source files for StarTouch Python SDK can be downloaded from the [Github repo](https://github.com/write-for-bug/startouch_python_sdk).

You can either clone the public repository:

```sh
git clone git://github.com/write-for-bug/startouch_python_sdk
```

Or download the [tarball](https://github.com/write-for-bug/startouch_python_sdk/tarball/master):

```sh
curl -OJL https://github.com/write-for-bug/startouch_python_sdk/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd startouch_python_sdk
uv pip install .
```
